ખાતા બુક (Firebase + Export + Mobile UI)

આ વેબસાઈટમાં Firebase Auth, Realtime Database, Excel Export, અને WhatsApp SMS ટેમ્પલેટ સામેલ છે.

1. Firebase project કેવી રીતે બનાવવું

  1. https://console.firebase.google.com પર જાઓ
  2. એક નવું project બનાવો (eg. khata-book)
  3. Authentication → Sign-in Method → Phone/Email ચાલુ કરો
  4. Realtime Database → Start in test mode → Enable કરો
  5. Project settings → General → Web App ઉમેરો → config કોપી કરો
  6. config.js ફાઇલમાં એ config પેસ્ટ કરો

2. Excel export માટે

SheetJS લાઇબ્રેરી દ્વારા .xlsx ફાઇલ export થાય છે

3. WhatsApp/SMS માટે

Twilio/Fast2SMS દ્વારા Reminder મોકલી શકાય છે. Code સામેલ છે.
